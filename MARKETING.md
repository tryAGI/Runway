# MARKETING.md

## Product
- One-liner: A .NET command-line tool that turns one prompt into a planned, multi-shot Runway short video.
- Category: developer tools / creative CLI

## Audience
- Persona: indie developers and creative engineers who already automate things from the terminal
- Awareness stage: solution-aware
- Pain points: manual storyboarding, juggling Runway dashboard tabs, scripting glue between text-to-video and ffmpeg, no way to keep a brand brief next to the project

## Goal
- Primary objective: activation
- Call to action: install with `dotnet tool install -g Runway.Cli` and run `runway short-video`
- Success metric: first successful `runway short-video` plan generated within 5 minutes of install

## Message
- Core value proposition: one command takes a scenario, plans the shots, generates the video, and concatenates the result — without leaving the terminal.
- Differentiators: marketing-brief aware planning, deterministic + Claude/Codex planners, ffmpeg concat fallback, AOT-trim safe, integrates with existing Runway API keys
- Proof points: ships as a NuGet dotnet tool, source-generated JSON, 60+ integration tests, sub-second deterministic planner

## Voice
- Tone: practical, technical, quietly confident
- Words to avoid: revolutionary, magic, game-changing, hack

## Format
- Platform: youtube-shorts
- Aspect ratio: 720:1280
- Length seconds: 30
- Captions on screen: yes
